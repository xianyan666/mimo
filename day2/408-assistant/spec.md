# 408考研学习助手 - 产品规格说明

## 1. 项目目标

构建一个基于CLI的408考研学习助手，帮助用户：

- 记录每日学习进度（科目、章节、时长、内容、问题）
- 跟踪整体学习状态（累计时长、科目分布、章节覆盖）
- 识别薄弱知识点（多因素评分机制）
- 生成阶段性复盘报告（趋势分析、遗忘风险、冲刺预测）
- 通过本地JSON文件维护长期学习记忆

---

## 2. 非目标

以下功能**不在**本项目范围内：

- 多用户支持
- 多设备同步/云端存储
- 真实LLM API集成（使用模拟AI总结）
- 图形界面（仅CLI）
- 自动计时功能
- 学习资源推荐
- 模拟考试/题目练习

---

## 3. 核心功能

### 3.1 submit - 提交学习记录

**交互式输入：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 科目 | 选择 | 是 | 数据结构/计算机组成原理/操作系统/计算机网络 |
| 章节 | 文本 | 是 | 具体章节名称 |
| 学习时长 | 数值 | 是 | 单位：小时（0-16） |
| 完成内容 | 文本 | 是 | 学习了什么 |
| 遇到的问题 | 文本 | 否 | 疑问或困难 |
| 自我评价 | 数值 | 是 | 理解度1-5分 |

**输出：**
- 确认信息
- AI模拟总结（模板生成）

---

### 3.2 today - 查看今日记录

**输出内容：**
- 今日所有学习记录列表
- 今日总学习时长
- 各科目时长分布
- 今日问题汇总

---

### 3.3 check - 检查整体进度

**输出内容：**

| 指标 | 说明 |
|------|------|
| 总学习时长 | 累计学习小时数 |
| 连续学习天数 | 最近连续提交记录的天数 |
| 科目占比 | 各科目学习时长百分比 |
| 章节覆盖率 | 已学习章节/总章节 |
| 高频疑问 | 出现次数最多的问题 |
| 薄弱知识点 | WeaknessScore ≥ 8的知识点 |
| 遗忘风险 | 超过7天未复习的内容 |
| 冲刺预测 | 基于当前进度的完成预测 |

---

### 3.4 review - 阶段性复盘报告

**输出内容：**
- 时间范围内的学习趋势
- 高频错误汇总
- 重复遗忘知识点
- 薄弱知识点详情及建议
- 距离目标的差距分析

---

## 4. 薄弱知识点算法

### 4.1 WeaknessScore 计算公式

```
WeaknessScore = (提问权重 × 提问次数) + 
                (重复权重 × 重复学习次数) + 
                (评价权重 × (5 - 自评理解度)) + 
                (遗忘权重 × 未复习天数)
```

### 4.2 权重配置

| 因素 | 权重 | 说明 |
|------|------|------|
| 提问次数 | 2.0 | 用户记录的问题次数 |
| 重复学习次数 | 1.5 | 同一知识点重复学习次数 |
| 自评理解度 | 1.0 | 5分制，取反计算 |
| 未复习天数 | 0.5 | 距离上次学习的天数 |

### 4.3 判定规则

- WeaknessScore < 4：掌握良好
- 4 ≤ WeaknessScore < 8：需要关注
- WeaknessScore ≥ 8：薄弱知识点，需重点复习

## 6. 边界条件处理

| 场景 | 处理方式 |
|------|----------|
| 输入为空 | 提示"输入不能为空，请重新输入" |
| 非408内容 | 提示"该内容不属于408范围，是否保存为备注" |
| 学习时长 < 0 | 提示"学习时长不能为负数" |
| 学习时长 > 16 | 提示"学习时长超过16小时，请确认"，需二次确认 |
| 同一天重复提交 | 采用追加模式，允许同一天多次记录 |
| 无历史记录 | check命令提示"暂无学习记录" |
| 记忆文件损坏 | 自动备份为memory_backup.json，重新创建 |
| 配置文件缺失 | 使用默认配置，提示用户运行初始化 |
| 章节不存在 | 提示用户选择已有章节或添加新章节 |

## 8. 命令行接口

```bash
# 初始化项目（创建配置文件和数据目录）
python cli.py init

# 提交学习记录
python cli.py submit

# 查看今日记录
python cli.py today

# 查看整体进度
python cli.py check

# 生成复盘报告
python cli.py review

# 查看配置
python cli.py config

# 修改配置
python cli.py config --set <key> <value>
```
