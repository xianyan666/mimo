## 五、核心算法测试用例（20条）

### TC-A01：WeaknessScore计算-基础公式验证
- **输入数据**：提问次数=2，重复学习=3次，平均评价=2分，未复习=10天
- **计算过程**：2.0×2 + 1.5×3 + 1.0×(5-2) + 0.5×10 = 4+4.5+3+5 = 16.5
- **预期结果**：WeaknessScore=16.5

### TC-A02：WeaknessScore计算-无提问
- **输入数据**：提问次数=0，重复学习=2次，平均评价=4分，未复习=3天
- **计算过程**：2.0×0 + 1.5×2 + 1.0×(5-4) + 0.5×3 = 0+3+1+1.5 = 5.5
- **预期结果**：WeaknessScore=5.5

### TC-A03：WeaknessScore计算-高评价低分
